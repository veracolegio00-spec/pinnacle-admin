# Pinnacle Admin

Panel administrativo desarrollado en React + Vite para la gestión de clientes, noticias y turnos del proyecto Pinnacle.

## Tecnologías

* React
* Vite
* Firebase Authentication
* GitHub

## Funcionalidades actuales

* Inicio de sesión mediante Firebase Authentication.
* Validación de usuarios administradores.
* Panel administrativo básico.
* Gestión de sesión.

## Instalación

Clonar el repositorio:

git clone https://github.com/veracolegio00-spec/pinnacle-admin.git

Instalar dependencias:

npm install

Ejecutar el proyecto:

npm run dev
